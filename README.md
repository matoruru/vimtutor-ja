# vimtutor-ja
日本語の説明で英語の練習用テキストが使えるvimtutor

## Install いんすとーる
1. 以下のスクリプトを実行する

       bash install.sh

1. 以下のコマンドを実行する
       
    
       cp ./vimtutor-ja.sh /usr/local/bin/vimtutor-ja
       chmod 755 /usr/local/bin/vimtutor-ja
       
       または
       
       sudo cp ./vimtutor-ja.sh /usr/local/bin/vimtutor-ja
       sudo chmod 755 /usr/local/bin/vimtutor-ja
    
## Usage　つかいかた
以下のコマンドを実行するだけ

    vimtutor-ja
